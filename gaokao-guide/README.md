# 上海高考指南

基于 2023-2025 上海高考全科真题考点与命题规律，为不同基础学生编写的全面、深度、可操作的备考指南。本仓库包含两份互补的指南：

- **《上海高考逆袭指南》**：面向不同基础学生的全面逆袭方法论，含 15 章内容、分级路径与冲刺计划。
- **《上海高考六科满分指南》**：聚焦语数英物化生六科的应试方法与题型导图，逐科拆解"考什么、怎么考、怎么应对"。

## 特色

- **基于真题**：所有考点分析基于 2023-2025 三年上海高考真题
- **逐题型精解**：每个题型给出"考点定位—命题意图—解题路径—易错点—满分模板"五段式拆解
- **命题人视角**：从命题流程、难度控制、区分度设计、反押题策略等角度揭秘
- **分级路径**：针对 400 分以下到 550+ 四个分数段给出不同逆袭路径
- **冲刺计划**：100/60/30/7 天四个时间维度的精细到天的计划表
- **心理建设**：考前心理调节、考场策略、突发情况应对预案

## 数据来源

- 2023-2025 上海高考全科真题（语数英物化生 + 等级考 + 春考）
- 真题详见同仓库 `shanghai-gaokao/` 目录
- 数据来源：上海市教育考试院 + 主流教育平台

## 适用人群

- 上海高三学生（含春考、秋考、等级考考生）
- 上海高二学生（提前规划）
- 家长（理解高考制度、配合孩子备考）
- 教师（了解命题趋势、调整教学策略）

### 满分指南特色

- **基于近三年真题**：所有考情分析来自 2023-2025 真题（语文"认可度"作文、数学解析几何压轴、物理"汽车智能化/氢同位素"主题、化学"福辛普利合成"、生物"CRB1 基因"等）
- **6 张题型思维导图**：每学科一张横向树状导图，中心学科 → 6 大题型 → 核心要点
- **可执行的方法论**：每个题型给出具体答题模板（物理论证"判断+依据+推理+结论"四段式、化学有机"逆推法"四步、生物遗传"五步法"）
- **避坑指南表格**：每章末尾用三列表格（陷阱类型 / 典型表现 / 规避方法）总结高频失分点
- **跨学科协同**：第八章整合六科时间规划，强调等级考（5 月）与统考（6 月）的节奏切换
