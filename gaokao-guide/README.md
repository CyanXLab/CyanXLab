## 特色

- **基于真题**：所有考点分析基于 2023-2025 三年上海高考真题
- **逐题型精解**：每个题型给出"考点定位—命题意图—解题路径—易错点—满分模板"五段式拆解
- **命题人视角**：从命题流程、难度控制、区分度设计、反押题策略等角度揭秘
- **分级路径**：针对 400 分以下到 550+ 四个分数段给出不同逆袭路径
- **冲刺计划**：100/60/30/7 天四个时间维度的精细到天的计划表
- **心理建设**：考前心理调节、考场策略、突发情况应对预案
