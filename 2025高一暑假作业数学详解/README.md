## 内容概览

本手册基于2025年高一数学暑假作业（共90页），涵盖五大模块：

1. **三角函数**（第1-5周）：任意角三角函数、恒等变换、解三角形、图象与性质、y=Asin(ωx+φ)
2. **平面向量**（第6周）：线性运算、数量积、模与夹角
3. **复数**（第7周）：概念、四则运算、几何意义
4. **数列**（第8周）：等差等比、求和方法、数学归纳法
5. **导数**（第9-10周）：导数概念、切线、单调性、极值最值、含参讨论

## 特色

- **难度分级**：★简单题（直接答案）、★★中档题（思路触发器）、★★★难题（完整解析）、★★★★压轴题（解析+模型+变式）
- **上海高考考频分析**：每章开头分析近5年上海高考命题特点与考频分布
- **认知科学学习方法**：融入间隔重复、主动检索、费曼技巧、交错练习等科学学习法
- **方法模型归纳**：难题附带通用解题模型与触发器，帮助举一反三
- **易错警示**：每章总结常见错误与避坑指南
- **提分策略**：每章给出针对性的备考建议

## 使用建议

1. 先独立完成暑假作业，再对照本手册查漏补缺
2. 遮住解析部分，尝试主动回忆解题过程
3. 难题（★★★及以上）用费曼技巧向他人讲解
4. 按周次复习，形成间隔重复闭环
